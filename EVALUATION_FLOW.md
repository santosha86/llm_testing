## Which Model is Used When?

| Screen | Which Models Used |
|--------|-------------------|
| **Configuration â†’ Test Connection** | Tests **ONE** model at a time (whichever you clicked) |
| **Evaluation â†’ Run Tests** | Uses **BOTH** models (baseline + target) |
| **Reports â†’ View Results** | Shows results from **BOTH** models |

---

## Summary

1. **You configure TWO models**: Baseline (reference) + Target (being tested)
2. **Both models answer the same questions** in parallel
3. **Results show BOTH answers** so you can compare
4. **Use Baseline** (OpenAI) as the "correct" standard
5. **Compare Target** (Ollama) against it to find gaps

---
